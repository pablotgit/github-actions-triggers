# github-actions-triggers
TAREA: "Automatización con Triggers de GitHub Actions"

GitHub Actions - Triggers
👤 Pablo Andres Tonelli

📌 Descripción

Repositorio con 6 workflows de GitHub Actions que demuestran distintos triggers.

⚙️ Workflows
Workflow Push

Se ejecuta al hacer commit en main.

Workflow PR

Se ejecuta al abrir un PR.

Workflow Issues

Se ejecuta al crear un issue.

Workflow Comentarios PR

Se ejecuta al comentar en un PR.

Workflow Manual

Ejecución manual con selección de nivel.

Workflow Programado

Se ejecuta automáticamente cada 5 minutos.
